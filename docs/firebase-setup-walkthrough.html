Step 8 of 8

Register Web App & Get Config

Click the gear icon ⚙️ (Project Settings) at the top of the sidebar. Scroll down to "Your apps" and click the web icon </>.

Register Web App

App nickname: timenest-web

After registering, you'll see this config object. Copy it!

Copy the config and paste it into the chat. It looks like this:

const firebaseConfig = { apiKey: "AIzaSyB.....................", authDomain: "timenest-app.firebaseapp.com", projectId: "timenest-app", storageBucket: "timenest-app.appspot.com", messagingSenderId: "123456789012", appId: "1:123456789:web:abc123def456" };
🎉 That's it! Once you paste the config here, I'll update your project files and everything will be connected.
